## 二、搬瓦工的价格体系为什么这么"乱"

其实搬瓦工的价格体系并不乱，只是它把"线路等级"和"配置大小"两个维度叠在一起卖，所以表面看着眼花。

**线路等级**决定你访问速度的稳定性，从低到高大概是这样：

1. **KVM 普通线路**：走 ChinaNet（AS4134）这种常规国际线路，晚高峰容易堵，但便宜。
2. **CN2 GT**：之前叫"CN2 套餐"，回程走 CN2 GT，比纯普通线路好一点，但 2019 年之后也开始堵了。现在搬瓦工基本把它并入 CN2 系列卖。
3. **CN2 GIA-E**：回程三网走 CN2 GIA（电信 AS4809）+ CMIN2（移动 AS58807）+ 联通优质线路（AS10099），这是搬瓦工的招牌，也是国内用户最该关注的档。
4. **香港/东京/新加坡/大阪 CN2 GIA**：物理机房就在亚洲，CN2 GIA 直连，延迟最低，但贵。

**配置大小**则是 1GB 到 64GB 内存、20GB 到 1280GB SSD 这种阶梯。

把这两个维度一组合，就变成了你看到的那一大堆套餐。理解了这点，下面那张全套餐表格你就能看明白了。
